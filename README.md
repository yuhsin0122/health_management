# health_management

我們的畢業專題  
Health Management App — Flutter 專案

---

## 專案介紹
這是一個以 Flutter 開發的健康管理應用程式，功能包含：
- 慢性病紀錄與追蹤
- 分析頁（血壓、血糖趨勢圖，支援日/週/月切換）
- 我的診所（收藏、回診倒數、加入 Google 行事曆）
- 藥物管理與提醒系統
